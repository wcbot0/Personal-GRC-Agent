# ISO/IEC 27001:2022 — Annex A Controls

**Source:** Annex A, ISO/IEC 27001:2022 (control text aligned with ISO/IEC 27002:2022)

93 controls grouped into four themes. Tag format: `ISO27001:A.<section>.<number>` — e.g. `ISO27001:A.5.15`.

Organizations must consider all controls; applicability is determined by risk assessment and documented in the Statement of Applicability (SoA).

## 2022 changes summary

| Change | Detail |
| --- | --- |
| Control count | 114 (2013) → 93 (2022) |
| Structure | 14 domains → 4 themes (Organisational, People, Physical, Technological) |
| New controls | 11 added (see table above) |
| Attributes | ISO 27002 adds control attributes (control type, security properties, cybersecurity concepts, operational capabilities, security domains) for risk-based mapping |
